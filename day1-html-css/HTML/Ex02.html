Basic HTML Tags

Every HTML document has the following things:

Headings: Used to provide headings to the document with tags from <h1>(largest) to <h6>(smallest).

Sections: Blocks that have replaced divs within the HTML document; created using <section> tag

Paragraphs: Similar to divs but used to present small amount of textual content within the div. Provided with a tag called <p>. Paragraphs cannot be nested but divs can be nested.

Spans: Inline content are called as spans; used with a tag called <span>.
